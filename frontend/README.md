# ⚡ PowerGuide

A community-driven power outage monitoring web app for **Dagupan City, Philippines**.

PowerGuide helps users track outages, monitor battery readiness, report blackouts, and locate charging hubs during emergencies.

---

## Features

- ⚡ Real-time outage monitoring
- 🗺️ Community outage reporting map
- 🔋 Battery tracker with alerts
- 💡 Smart energy-saving tips
- 🔌 Charging hub locator
- 🔔 Power outage notifications

---

## Tech Stack

- **HTML5**
- **Tailwind CSS**
- **Vanilla JavaScript**
- **Font Awesome**
- **Battery Status API**
- **Google Maps Embed API**
